# 命令行

提供 **Perigon**命令行工具。

## new

`new`命令用来创建一个新的解决方案，与使用`Studio`图形化界面创建是相同的效果。

```pwsh
perigon new <name>
```

创建解决方案的流程：

1. 选择数据库类型：Sqlserver、PostgreSql
2. 选择缓存类型
3. 选择前端集成
4. 输入目录，默认为当前目录
5. 确认并运行

## studio

`studio`命令用来启动Dashboard，大多数操作都可以通过Dashboard来完成。

```pwsh
perigon studio
```

## generate

提供代码生成相关的命令，别名`g`。

目前支持客户端请求服务的生成，如:

```pwsh
perigon g request https://localhost:17001/swagger/v1/swagger.json ./src/services -t angular
```

你可以使用`perigon g request -h`来查看帮助信息，如:
```pwsh
DESCRIPTION:
生成客户端请求服务及模型

USAGE:
    perigon generate request <path|url> <outputPath> [OPTIONS]

EXAMPLES:
    perigon generate request ./openapi.json ./src/services -t angular

ARGUMENTS:
    <path|url>      Local path or url, support json format
    <outputPath>    The output path

OPTIONS:
                        DEFAULT
    -h, --help                     Prints help information
    -t, --type          angular    Support types: csharp/angular/axios, default: angular
    -m, --only-model    false      Only generate model files

```
