# 概述

借助AI编程，已经是验证过的提高编码效率的方式。目前在Perigon中，提供MCP工具和自定义MCP来帮助开发者更高效地编写代码。

## 现实问题

不同的模型提供商，不同的IDE，使用AI的方式各不相同，而且现在还在不断的发展和变化，包括但不限于以下方式或工具：

- CLI
- skills
- prompts
- instructions
- MCP
- Agent
- Plan
- Cloud

从工程角度来讲，各家有各家的实现方案，不同的技术对应不同的场景。

对于.NET开发者，我将优先探索和总结使用`Visual Studio`和`Visual Studio Code`中AI编程的最佳实践。


在模板项目中，你会看到包含以下几种文件

- Copilot instructions
- MCP
- skills

你可以根据自己的实际需求，选择合适的方式来使用AI编程。
