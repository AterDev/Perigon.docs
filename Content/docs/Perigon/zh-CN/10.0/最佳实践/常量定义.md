# 常量定义

在项目中，我们经常会定义一些常量或静态变量来存储不变的数据。为了提高代码的可读性和维护性，建议将这些常量集中管理。

## 共享常量

在`Share`程序集的`Constants`目录下，是专门用来定义共享常量或静态变量的地方。

## 其他常量

如果某些常量仅在特定的模块或服务中使用，应该将它们定义在相应的模块或服务中，而不是放在共享常量目录下。

## 扩展方法

在基础类库`Ater.Common`中已经使用了`AppConst`来定义一些常量。如果仍然想使用`AppConst`，可以通过扩展方法的方式来实现。

在C#14中，我们可以在`Share/Constants`目录下创建一个新的静态类`AppExtensions`，专门用来扩展应用级别的变量和方法，如：

```csharp
public static class AppExtensions
{
    extension(Ater.Common.AppConst)
    {
        public static string AIAgent => "AIAgent";
    }
}
```

然后我们就可以在代码中直接使用`AppConst.AIAgent`来访问这个变量。

> [!TIP]
> 如果使用时提示找不到定义的变量或扩展方法，尝试手动添加命名空间的引用。
